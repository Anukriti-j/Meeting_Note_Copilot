# Meeting Notes Copilot — frontend

React + TypeScript + Vite UI for the meeting-notes-copilot backend. See the
[project root README](../README.md) for setup and how to run the whole app —
the short version is `./start.sh` from the project root, which starts this
alongside the FastAPI backend.

Pages: Dashboard, Ingest, Ask (streaming, over WebSocket), Agent (also shows the
calendar events it creates), Action Items, Logs.
